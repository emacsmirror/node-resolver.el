# node-resolver-mode.el

## Usage

`npm install -g node-resolver`

Then in your `init.el`

``` elisp
(load-file "~/Code/node-resolver-mode/node-resolver-mode.el")
(add-hook 'js3-mode-hook 'node-resolver-mode)
```
